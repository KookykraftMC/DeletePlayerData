# DeletePlayerData
Easy ingame command to delete player.dat files.

## Download
All builds for my plugins can be found at this link: http://kaikk.net/mc/

###Installation
Just drop the jar to plugins folder.

###Usage
/deleteplayerdata (PlayerName)
